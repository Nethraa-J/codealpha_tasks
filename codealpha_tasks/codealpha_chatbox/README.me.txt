Simple Python Chatbox
A simple terminal-based chatbox built using Python. It simulates basic conversation logic and demonstrates input handling, loops, and conditional statements.

Features
Interactive terminal conversation
Handles user input and responses
Exit command to end chat
Beginner-friendly Python project
How to Run
Make sure Python is installed
Download or clone this repository
Run the file:
python chatbox.py

Example
You: hello
Bot: Hi! How can I help you?

You: bye
Bot: Goodbye!
