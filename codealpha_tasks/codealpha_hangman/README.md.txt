🎮 Hangman Game (Python)
📌 Overview
A command-line based Hangman game built using Python.
Developed as part of the CodeAlpha Internship to demonstrate input validation, game logic handling, and clean user interaction.

🚀 Features
Random word selection
Input validation (only single alphabet allowed)
Prevents duplicate guesses
Displays guessed letters
Limited attempts system
Clean win/lose game-ending message

🛠️ Technologies Used
Python 3

▶️ How to Run
Make sure Python is installed.
Clone the repository: git clone https://github.com/your-username/hangman-game-python.git
Navigate to the folder: cd hangman-game-python
Run the program: python hangman.py
